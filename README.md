# Real-Time Location Monitoring and Geofence Alert System for Alzheimer's

## Overview

This project provides real-time location tracking and geofence-based alert functionality for Alzheimer's patients. The system helps caregivers monitor patient movement and receive alerts whenever the patient crosses predefined safe boundaries. The solution improves patient safety and enables continuous monitoring through IoT-based technologies.

## Features

* Real-time GPS location tracking
* Geofence boundary monitoring
* Alert generation when safe zones are crossed
* Patient safety and caregiver assistance
* IoT-based monitoring system

## Technologies Used

* Arduino
* GPS Module
* Embedded Systems
* Internet of Things (IoT)

## Applications

* Alzheimer's patient monitoring
* Elderly care systems
* Smart healthcare solutions
* Personal safety tracking
